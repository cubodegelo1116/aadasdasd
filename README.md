# RXT ADMIN v1.0

Script de administração para Roblox com diversos comandos utilitários.

## 📋 Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `cmds` | Abre esta lista de comandos |
| `prefix [novo]` | Muda o prefixo |
| `vprefix` | Mostra o prefixo atual |
| `popup true` | Ativa os popups |
| `popup false` | Desativa os popups |
| `fly [velocidade]` | Ativa o modo voo |
| `unfly` | Desativa o modo voo |
| `noclip` | Ativa o noclip |
| `clip` | Desativa o noclip |
| `speed [numero]` | Altera a velocidade |
| `jump [numero]` | Altera o pulo |
| `loopspeed [numero]` | Loop da velocidade |
| `unloopspeed` | Desativa loop da velocidade |
| `loopjump [numero]` | Loop do pulo |
| `unloopjump` | Desativa loop do pulo |
| `infjump` | Ativa pulo infinito |
| `uninfjump` | Desativa pulo infinito |
| `goto [nome]` | Teleporta até o jogador |
| `bring [nome]` | Puxa o jogador até você |
| `view [nome]` | Observa o jogador |
| `unview` | Para de observar |
| `tptool` | Dá uma tool de teleporte |
| `untptool` | Remove a tool de teleporte |
| `esp` | Ativa o ESP (wallhack) |
| `unesp` | Desativa o ESP |
| `walkfling` | Ativa walkfling |
| `unwalkfling` | Desativa walkfling |
| `executor` | Abre o executor de scripts |
| `rejoin` | Reentra no servidor |
| `reset` | Respawna o personagem |
| `float` | Ativa o float (Q desce, E sobe) |
| `unfloat` | Desativa o float |
| `sit` | Senta o personagem |

## 🎮 Configurações

- **Prefixo padrão**: `/`
- **Popups**: Ativados por padrão

## 🚀 Uso

1. Execute o script `adminscript.lua` no seu executor Roblox
2. Digite os comandos usando o prefixo configurado
3. Exemplo: `/fly 50` para ativar o voo com velocidade 50

## ⚙️ Funcionalidades

- ✈️ Modo voo com controle de velocidade
- 🚫 Noclip (atravessar paredes)
- ⚡ Modificação de velocidade e pulo
- 📍 Teleportação entre jogadores
- 👁️ ESP (visualizar jogadores através de paredes)
- 🎯 Walkfling e Float
- 💻 Executor de scripts integrado

## ⚠️ Aviso

Este script é apenas para fins educacionais. Seu uso em servidores não autorizados pode resultar em banimento.
