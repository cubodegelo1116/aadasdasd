# RXT ADMIN 

Script de admin para Roblox com diversos comandos utilitários.

## 📋 Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `cmds` | Abre a lista de comandos |
| `prefix [novo]` | Muda o prefixo |
| `vprefix` | Mostra o prefixo atual |
| `popup true/false` | Ativa ou desativa os popups |
| `executor` | Abre o executor de scripts |
| `rspy` | Carrega o SimpleSpy V3 |
| `invisible` / `invis` | Te deixa invisível |
| `visible` / `vis` / `uninvisible` | Te deixa visível novamente |
| `invfly [velocidade]` | Ativa invfly (invisível + fly + noclip) |
| `uninvfly` | Desativa invfly |
| `rejoin` | Reentra no servidor |
| `reset` | Respawna o personagem |
| `sit` | Senta/levanta o personagem |
| `fly [velocidade]` | Ativa o modo voo |
| `unfly` | Desativa o modo voo |
| `noclip` | Ativa o noclip |
| `clip` | Desativa o noclip |
| `speed [numero]` | Altera a velocidade de movimento |
| `unspeed` | Volta a velocidade normal |
| `loopspeed [numero]` | Ativa loop de velocidade |
| `unloopspeed` | Desativa loop de velocidade |
| `jump [numero]` | Altera a força do pulo |
| `unjump` | Volta o pulo ao normal |
| `loopjump [numero]` | Ativa loop de pulo |
| `unloopjump` | Desativa loop de pulo |
| `infjump` | Ativa pulo infinito |
| `uninfjump` | Desativa pulo infinito |
| `goto [nome]` | Teleporta até o jogador |
| `bring [nome]` | Puxa o jogador até você |
| `view [nome]` | Observa a câmera de outro jogador |
| `unview` | Para de observar |
| `tptool` | Dá uma ferramenta de teleporte |
| `untptool` | Remove a ferramenta de teleporte |
| `esp` | Ativa o ESP (wallhack) |
| `unesp` | Desativa o ESP |
| `walkfling` | Ativa walkfling |
| `unwalkfling` | Desativa walkfling |
| `float [valor]` | Ativa float (controle com Q e E) |
| `unfloat` | Desativa o float |

## 🎮 Configurações

- **Prefixo padrão**: `/`
- **Popups**: Ativados por padrão

## 🕹️ Controles

### Fly / Invfly
- **W** - Mover para frente
- **S** - Mover para trás
- **A** - Mover para esquerda
- **D** - Mover para direita

### Float
- **Q** - Descer
- **E** - Subir
